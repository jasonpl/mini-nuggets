mini-nuggets
============

## alert: this is a work in progress


Description
===========
A chicken defense map for smaller teams (2~3 players).
This map is an interpretation of KaiserJ's popular [Chicken Nuggets map](http://zero-k.info/maps/Detail/7231) for the [Spring engine](http://springrts.com/).


Tools of the trade
==================

- mspaint
- PowerPoint
